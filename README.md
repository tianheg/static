# jsDelivr CDN static

### Usage:

```text
https://cdn.jsdeliver.net/gh/[user]/[repo]@[branch]/[something].[jpg/css/js]
```

### Demo:

```text
https://cdn.jsdelivr.net/gh/tianheg/static@main/img/xxx.jpg
```

### Intro

 分支 | 用处 | 链接
 --- | --- | ---
 main | 默认分支 | <https://github.com/tianheg/static>
